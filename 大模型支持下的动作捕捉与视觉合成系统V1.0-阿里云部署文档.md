## 二、阿里云资源规划

### 2.1 地域与可用区选择

**推荐配置：**
- **地域**：华东1（杭州）或 华东2（上海）
- **可用区**：多可用区部署（如杭州可用区G + 可用区H）
- **网络**：专有网络VPC

**选择理由：**
- 靠近用户群体，降低网络延迟
- GPU实例库存充足
- 支持多可用区高可用部署

### 2.2 网络架构设计

```
VPC: 192.168.0.0/16
│
├── 交换机-公网 (192.168.1.0/24) - 可用区G
│   ├── NAT网关
│   ├── SLB负载均衡 (公网)
│   └── Web服务器-1 (192.168.1.10)
│
├── 交换机-应用 (192.168.2.0/24) - 可用区G
│   ├── 应用服务器-1 (192.168.2.10)
│   ├── 应用服务器-2 (192.168.2.11)
│   └── AI网关服务器 (192.168.2.20)
│
├── 交换机-数据 (192.168.3.0/24) - 可用区G
│   ├── RDS MySQL主库
│   └── Redis主节点
│
├── 交换机-GPU (192.168.4.0/24) - 可用区G
```

## 三、GPU服务器部署

### 3.1 创建GPU实例

**步骤1：申请GPU配额**

登录阿里云控制台 → 配额中心 → 通用配额 → 申请ECS GPU实例配额

**步骤2：创建GPU实例**

```bash
# 使用阿里云CLI创建GPU实例（示例）
aliyun ecs CreateInstance \
  --RegionId cn-hangzhou \
  --ZoneId cn-hangzhou-g \
  --InstanceType ecs.gn7i-c16g1.4xlarge \
  --ImageId ubuntu_22_04_x64_20G_alibase_20230613.vhd \
  --SystemDiskCategory cloud_essd \
  --SystemDiskSize 200 \
  --VpcId vpc-xxx \
  --VSwitchId vsw-xxx \
  --SecurityGroupId sg-xxx \
  --InstanceName mocap-gpu-01 \
  --KeyPairName mocap-key \
  --InternetChargeType PayByTraffic \
  --InternetMaxBandwidthOut 100
```

**推荐实例规格：**

| 场景 | 实例规格 | GPU | 显存 | CPU | 内存 |
|------|----------|-----|------|-----|------|
| 开发测试 | gn7i-vws | 1×A10 | 24GB | 4核 | 15GB |
| 生产环境 | gn7i.8xlarge | 1×A100 | 80GB | 32核 | 120GB |
| 大规模生产 | gn7.26xlarge | 8×A100 | 640GB | 104核 | 800GB |

### 3.2 CUDA环境配置

**步骤1：登录GPU服务器**

```bash
ssh -i ~/.ssh/mocap-key root@<GPU服务器公网IP>
```

**步骤2：安装NVIDIA驱动**

```bash
# 更新系统
apt-get update && apt-get upgrade -y

# 安装必要工具
apt-get install -y build-essential dkms

# 查看推荐驱动
ubuntu-drivers devices

# 安装推荐驱动（自动）
ubuntu-drivers autoinstall

# 或手动安装特定版本
apt-get install -y nvidia-driver-535

# 重启服务器
reboot
```

**步骤3：验证驱动安装**

```bash
# 重启后登录，验证驱动
nvidia-smi

# 预期输出：
# +---------------------------------------------------------------------------------------+
# | NVIDIA-SMI 535.104.05             Driver Version: 535.104.05   CUDA Version: 12.2     |
# |-----------------------------------------+----------------------+----------------------+
# | GPU  Name                 Persistence-M | Bus-Id        Disp.A | Volatile Uncorr. ECC |
# | Fan  Temp   Perf          Pwr:Usage/Cap |         Memory-Usage | GPU-Util  Compute M. |
# |                                         |                      |               MIG M. |
# |=========================================+======================+======================|
# |   0  NVIDIA A100-SXM4-80GB          Off | 00000000:00:05.0 Off |                    0 |
# | N/A   35C    P0              44W / 400W |      0MiB / 81920MiB |      0%      Default |
# |                                         |                      |             Disabled |
# +-----------------------------------------+----------------------+----------------------+
```

**步骤4：安装CUDA Toolkit**

```bash
# 下载CUDA 12.1（推荐版本）
wget https://developer.download.nvidia.com/compute/cuda/12.1.0/local_installers/cuda_12.1.0_530.30.02_linux.run

# 运行安装程序
sh cuda_12.1.0_530.30.02_linux.run --silent --toolkit --installpath=/usr/local/cuda-12.1

# 配置环境变量
echo 'export PATH=/usr/local/cuda-12.1/bin:$PATH' >> ~/.bashrc
echo 'export LD_LIBRARY_PATH=/usr/local/cuda-12.1/lib64:$LD_LIBRARY_PATH' >> ~/.bashrc
source ~/.bashrc

# 验证CUDA安装
nvcc --version
```

**步骤5：安装cuDNN**

```bash
# 下载cuDNN（需要NVIDIA开发者账号）
# 从 https://developer.nvidia.com/cudnn 下载对应CUDA 12.x的版本

# 解压并安装
tar -xvf cudnn-linux-x86_64-8.9.7.29_cuda12-archive.tar.xz
cd cudnn-linux-x86_64-8.9.7.29_cuda12-archive

# 复制文件到CUDA目录
cp include/cudnn*.h /usr/local/cuda-12.1/include/
cp lib/libcudnn* /usr/local/cuda-12.1/lib64/
chmod a+r /usr/local/cuda-12.1/include/cudnn*.h /usr/local/cuda-12.1/lib64/libcudnn*

# 验证cuDNN安装
cat /usr/local/cuda-12.1/include/cudnn_version.h | grep CUDNN_MAJOR -A 2
```

### 3.5 创建系统服务

**步骤1：创建启动脚本**

```bash
cat > /opt/mocap-system/start_service.sh << 'EOF'
#!/bin/bash
# 大模型支持下的动作捕捉与视觉合成数据系统 V1.0 服务启动脚本

set -e

echo "========================================="
echo "启动 大模型支持下的动作捕捉与视觉合成数据系统 V1.0 服务"
echo "========================================="

# 激活conda环境
source /opt/miniconda3/etc/profile.d/conda.sh
conda activate mocap

# 设置环境变量
export PYTHONPATH=/opt/mocap-system:$PYTHONPATH
export CUDA_VISIBLE_DEVICES=0
export MOCAP_CONFIG=/opt/mocap-system/config/model_paths.yml

# 进入项目目录
cd /opt/mocap-system

# 检查GPU状态
echo "检查GPU状态..."
nvidia-smi

# 启动AI推理服务（FastAPI）
echo "启动AI推理服务..."
python -m uvicorn apps.mocap.service:app \
    --host 0.0.0.0 \
    --port 8000 \
    --workers 1 \
    --log-level info &

# 记录PID
echo $! > /var/run/mocap-service.pid

echo "服务启动完成，监听端口: 8000"
echo "API文档地址: http://<服务器IP>:8000/docs"
EOF

chmod +x /opt/mocap-system/start_service.sh
```

**步骤2：创建systemd服务**

```bash
cat > /etc/systemd/system/mocap-gpu.service << 'EOF'
[Unit]
Description=大模型支持下的动作捕捉与视觉合成数据系统 V1.0 GPU服务
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/opt/mocap-system
Environment="PATH=/opt/miniconda3/envs/mocap/bin:/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin"
Environment="CUDA_VISIBLE_DEVICES=0"
Environment="PYTHONPATH=/opt/mocap-system"
ExecStart=/opt/mocap-system/start_service.sh
ExecStop=/bin/kill -SIGTERM $MAINPID
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
EOF

# 重新加载systemd
systemctl daemon-reload

# 启用服务
systemctl enable mocap-gpu.service

# 启动服务
systemctl start mocap-gpu.service

# 查看服务状态
systemctl status mocap-gpu.service
```

- **实例规格**：ECS c7.2xlarge（8核16G）或更高
- **操作系统**：Alibaba Cloud Linux 3 / CentOS 8 / Ubuntu 22.04
- **系统盘**：ESSD PL0 100GB
- **数据盘**：ESSD PL1 500GB

**创建步骤：**

```bash
# 使用阿里云CLI创建应用服务器
aliyun ecs CreateInstance \
  --RegionId cn-hangzhou \
```

```bash
        """使用VLM增强关键点检测"""
        # 识别低置信度关键点
        low_conf_mask = confidence < 0.3
        
        if not np.any(low_conf_mask):
            return keypoints, confidence
        
        # 使用视觉大模型进行遮挡区域修复
        # 这里可以集成GPT-4V或其他VLM进行增强
        
        return keypoints, confidence

# 全局VLM服务实例
vlm_service = VLMService()
EOF
```

### 6.3 AIGC引擎部署

```bash
cat > /opt/mocap-system/easymocap/aigc/service.py << 'EOF'
"""
大模型支持下的动作捕捉与视觉合成数据系统 V1.0 - AIGC服务模块
提供AI生成内容功能，包括视角合成、场景生成等
"""

import torch
import numpy as np
from typing import List, Dict

class AIGCService:
    """AI生成内容服务"""
    
    def __init__(self):
        self.device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
        self._init_models()
    
    def _init_models(self):
        """初始化AIGC模型"""
        # Stable Diffusion用于场景生成
        try:
            from diffusers import StableDiffusionPipeline
            self.sd_pipeline = StableDiffusionPipeline.from_pretrained(
                "/opt/mocap-system/models/stable-diffusion-v1-5",
                torch_dtype=torch.float16
            ).to(self.device)
            print("[AIGC] Stable Diffusion加载成功")
        except Exception as e:
            print(f"[AIGC] Stable Diffusion加载失败: {e}")
            self.sd_pipeline = None
        
        # ControlNet用于姿态控制生成
        try:
            from diffusers import ControlNetModel, StableDiffusionControlNetPipeline
            controlnet = ControlNetModel.from_pretrained(
                "/opt/mocap-system/models/controlnet-openpose",
                torch_dtype=torch.float16
            )
            self.controlnet_pipeline = StableDiffusionControlNetPipeline.from_pretrained(
                "/opt/mocap-system/models/stable-diffusion-v1-5",
                controlnet=controlnet,
                torch_dtype=torch.float16
            ).to(self.device)
            print("[AIGC] ControlNet加载成功")
        except Exception as e:
            print(f"[AIGC] ControlNet加载失败: {e}")
            self.controlnet_pipeline = None
    
    def generate_scene(self, prompt: str, width: int = 512, height: int = 512) -> np.ndarray:
        """生成虚拟场景"""
        if self.sd_pipeline is None:
            return None
        
        image = self.sd_pipeline(
            prompt,
            width=width,
            height=height,
            num_inference_steps=50
        ).images[0]
        
        return np.array(image)
    
    def generate_with_pose(self, pose_image: np.ndarray, prompt: str) -> np.ndarray:
        """基于姿态生成图像"""
        if self.controlnet_pipeline is None:
            return None
        
        from PIL import Image
        pose_pil = Image.fromarray(pose_image)
        
        image = self.controlnet_pipeline(
            prompt,
            pose_pil,
            num_inference_steps=50
        ).images[0]
        
        return np.array(image)

# 全局AIGC服务实例
aigc_service = AIGCService()
EOF
```

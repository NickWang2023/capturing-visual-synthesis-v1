# 致业电子 LOGO 放置说明

## LOGO 文件

请将致业电子的LOGO图片放置在此目录下，文件命名为：

```
zecn_logo.png
```

## LOGO 规格建议

- **格式**: PNG (支持透明背景)
- **尺寸**: 建议宽度 800px 以上
- **比例**: 保持原始比例
- **背景**: 透明或白色背景

## 引用方式

在文档中引用LOGO：

```markdown
<div align="center">
    <img src="doc/assets/zecn_logo.png" width="40%">
    <h3>致业电子 ZECN</h3>
</div>
```

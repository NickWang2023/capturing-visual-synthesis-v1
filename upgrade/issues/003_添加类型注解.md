# Issue #3: 添加类型注解

## 📋 问题描述

当前代码缺少 Python 类型注解，影响代码可维护性和 IDE 支持。

## 🎯 目标

- [ ] 为核心函数添加类型注解
- [ ] 添加类型检查工具（mypy）
- [ ] 配置类型检查规则

## 📝 具体修改

### 1. pipeline/basic.py

```python
# 修改前
def multi_stage_optimize(body_model, params, kp3ds, kp2ds=None, bboxes=None, Pall=None, weight={}, cfg=None):
    # ...

# 修改后
from typing import Optional, Dict, Any
import numpy as np

def multi_stage_optimize(
    body_model: Any,
    params: Dict[str, Any],
    kp3ds: np.ndarray,
    kp2ds: Optional[np.ndarray] = None,
    bboxes: Optional[np.ndarray] = None,
    Pall: Optional[np.ndarray] = None,
    weight: Optional[Dict[str, float]] = None,
    cfg: Optional[Any] = None
) -> Dict[str, Any]:
    # ...
```

### 2. 需要添加类型注解的文件

- `easymocap/pipeline/basic.py`
- `easymocap/pipeline/config.py`
- `easymocap/dataset/` 目录下的所有文件
- `easymocap/smplmodel/` 目录下的所有文件
- `easymocap/estimator/` 目录下的所有文件

### 3. mypy 配置

创建 `mypy.ini` 文件：

```ini
[mypy]
python_version = 3.10
warn_return_any = True
warn_unused_configs = True
disallow_untyped_defs = True
check_untyped_defs = True

[mypy-spconv.*]
ignore_missing_imports = True

[mypy-chumpy.*]
ignore_missing_imports = True
```

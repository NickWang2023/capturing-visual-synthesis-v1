### 2. pytest配置

创建 `pytest.ini`：

```ini
[pytest]
testpaths = tests
python_files = test_*.py
python_classes = Test*
python_functions = test_*
markers =
    unit: 单元测试
    integration: 集成测试
    performance: 性能测试
    slow: 慢速测试
addopts = 
    -v
    --tb=short
    --cov=easymocap
    --cov-report=html
    --cov-report=xml
```
